# AZZRO - Fashion Marketplace

## 🚀 Project Overview
**AZZRO** is a premium, multi-vendor fashion eCommerce platform designed for the Indian market. 
Built with **Next.js 15**, **Tailwind CSS v4**, and **TypeScript**.

## 🎨 Design System
- **Fonts**: Outfit (Headings), Inter (Body)
- **Colors**: Jet Black (`#1A1A1A`), Pure White, Vibrant Coral (`#FF3F6C`)
- **Aesthetic**: Glassmorphism, Micro-animations, Clean Layouts.

## 🛠 Tech Stack
- **Framework**: Next.js 15 (App Router)
- **Styling**: Tailwind CSS v4
- **State**: React Hooks / Context (Planned: Zustand)
- **Icons**: Lucide React

## 📂 Structure
- `app/` - App Router pages and layouts.
- `components/` - Reusable UI components (To be created).
- `lib/` - Utilities and helpers.
- `public/` - Static assets.

## 🏃‍♂️ Getting Started
1. Install dependencies: `npm install`
2. Run dev server: `npm run dev`
3. Build for production: `npm run build`
